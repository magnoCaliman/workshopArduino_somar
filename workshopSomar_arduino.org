* 1 - BLINK
- Copiar e colar o código abaixo em uma janela nova da Arduino IDE
- Selecionar a placa e porta correta em Tools -> Board & Tools -> Port
- Salvar o arquivo
- Fazer upload

** CÓDIGO
#+BEGIN_SRC c
int pinoLed = 13;

void setup()
{
  pinMode(pinoLed, OUTPUT);
}

void loop()
{
  digitalWrite(pinoLed, HIGH);
  delay(1000);
  digitalWrite(pinoLed, LOW);
  delay(500);  
}
#+END_SRC
